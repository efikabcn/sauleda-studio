Este botón es el corazón del modelo: cada sesión acaba en muestra física + lead cualificado para el toldista. Sauleda vende metros; el toldista gana el cliente.

Cómo leer esta maqueta: es una simulación ilustrativa para las entrevistas de validación y la presentación interna — los tejidos son aproximaciones de color, no reproducciones fieles. En el producto real, la carta completa Sauleda se renderiza desde escaneos calibrados de fábrica, el visualizador se incrusta en la web de cada toldista con un fragmento de código («embed»), y cada diseño compartido lleva la referencia exacta del tejido. Modelo de referencia: Dickson Designer Pro (grupo Glen Raven).
